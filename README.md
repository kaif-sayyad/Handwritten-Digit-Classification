# Handwritten Digit Classification using CNN and RNN

## Overview
This project implements Handwritten Digit Classification on the MNIST dataset using:

1. Convolutional Neural Network (CNN)
2. Recurrent Neural Network (RNN)

The performance of both models is compared using accuracy and confusion matrix.

## Dataset
MNIST Dataset
- 70,000 grayscale images
- Image Size: 28 × 28
- Classes: Digits 0–9

## Technologies Used
- Python
- PyTorch
- NumPy
- Matplotlib
- Scikit-Learn

## Results

### CNN
- Accuracy: ~98–99%

### RNN
- Accuracy: ~95–97%

## Conclusion
CNN performs better than RNN for image classification because it efficiently captures spatial features from images.
